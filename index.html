INTRODUCCION

En este espacio encontrara documentación que le permitirá fortalecer el conocimiento con el uso de herramientas de diseño como es Git, Github, html5, css3. Se pone a disposición lecturas con conceptos básicos, videos en los que podrá crear un repositorio y los link de descarga de los programas. De esta forma usted contara con las bases necesarias para desarrollar un proyecto de diseño.

Objetivo general

Realizar un diseño en el que se registre información básica que permita facilitar el conocimiento de herramientas enfocadas al desarrollo web (GIT, GITHUB, HTML5 y CSS3)

Objetivo específicos

Explicar que función cumple GIT,GITHUB,HTML5 y CSS3 como parte fundamental del diseño

Ofrecer alternativas que permitan incentivar el uso de herramientas del diseño web por medio de videos y textos de fácil compresión.

Documentar la página web con información veraz dinámica y fácil de comprender el uso de herramientas de diseño.
